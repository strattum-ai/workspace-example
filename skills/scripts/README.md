# Scripts

Coloque aqui os scripts Python referenciados nas suas skills.

Cada script deve ser referenciado nas instruções da skill pelo caminho relativo, ex: `scripts/meu_script.py`.

## Exemplo de referência numa skill

```
Execute o script `scripts/portfolio_insights.py` com o argumento `entity_id`.
```

## Boas práticas

- Um script por skill (ou por operação atômica)
- Nomeie com snake_case: `portfolio_insights.py`
- Declare dependências em `/workspace/skills/requirements.txt`
